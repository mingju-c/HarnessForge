# API-Bank Eval Bench

This directory contains API-Bank `apis/`, `init_database/`, `test-data/`, and DeepAgent-style `lv1-lv2-samples/`. The wrapper exports `API_BANK_APIS_DIR` and `API_BANK_DATABASE_DIR` to local paths.

API-Bank is local/offline in this repository. No external API key is required; only the model endpoint variables such as `OPENAI_BASE_URL` and `OPENAI_API_KEY` are needed for inference.
